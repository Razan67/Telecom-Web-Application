# Telecom-Web-Application
QuickCharge App, a web application designed for a telecom company. The app's core function is to allow end-users to recharge their prepaid balance (via voucher or online payment) and provide a comprehensive administrative interface.

## QuickCharge App - Telecom Web Application Development

# Overview

This project is the Software Design Document (SDD) and associated design artifacts for the "QuickCharge App," a proposed web application for a telecom company. The application is designed to allow end-users to manage their accounts, recharge their balance via vouchers or online payment, and provide administrative tools for system management.

This repository serves as the documentation hub for the design phase of the application development lifecycle.

# Project Details

•
Project Name: QuickCharge App (Telecom Web Application)

•
Purpose: To provide a robust platform for end-users to recharge their prepaid balance and for administrators to manage the system and generate reports.

•
Documentation: The full project specification and design rationale are contained in the Software Design Document (SDD).

# Design Artifacts

The design of the QuickCharge App is fully documented through a series of UML and design diagrams, which are located in the assets/ folder.
| Diagram | Purpose | File Name |
